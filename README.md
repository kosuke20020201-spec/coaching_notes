# Session OS

パーソナルジムとボイトレで聞いたことを、あとから見返すためのシンプルなレッスンメモです。

細かく管理するより、読み返したときに「そうやったわ」と思い出せることだけを残す想定です。

## 使い方

`index.html` をブラウザで開くだけで使えます。

GitHub PagesなどのWebサーバー上に置くと、PWAとしてホーム画面に追加できます。オフライン用の `manifest.json` と `sw.js` も含めています。

記録はブラウザの `localStorage` に保存されます。同じブラウザで開けば前回の内容が残ります。

## 主な機能

- ジム、ボイトレ、横断メモの切り替え
- メモ履歴、検索、フィルタ
- 入力されたタグを自動でフォルダ化
- タグフォルダを開いて、そのタグのメモだけ表示
- タグフォルダ内で新規メモを作ると、そのタグを自動付与
- 新規メモ、保存、削除
- タイトル、日付、本文、タグだけの簡単入力
- スクロール中も新規メモボタンを上部に固定
- PWA対応

固定タグはありません。メモに書いたタグだけがフォルダとして表示されます。
